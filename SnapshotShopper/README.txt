How to use the app:
>Adding an item: Click the camera button to take a picture. Enter the name of 
		       the item and press the Add button to add the item to the list. 

>Checking off an item: Click anywhere where the desired item is to check the item.

>Editing the item entry: Long-click the item to open Edit Item page.
                         You can change the item name and the associated picture here.
                         Click Apply Changes button to reflect changes onto the list
	                 (Not implemented yet)
			 Press the back button on Android to go back without applying 
			 changes.

>Deleting one item entry: Open Edit item page with a desired item to be deleted and 
			  click Delete This Item button. (Not implemented yet)

>Deleting the entire list: On the list page, click Clear List button.
			   You will be asked for a confirmation before all items in the list
			   gets deleted.
